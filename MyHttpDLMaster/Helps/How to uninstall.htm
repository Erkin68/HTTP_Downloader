This program will work without any installing package.

 

Remember following when we will want uninstall this program:

 

  Path of the configuration file-‘conf.bin’ and history file ‘history.bin’ founded via instruction:

 

  SHGetFolderPath(NULL,CSIDL_APPDATA,NULL,SHGFP_TYPE_CURRENT,s);

 

  This file can be placed in path  “C:\Users\UserName\AppData\Roaming\myHttpDL\conf.bin” for example.

 

  You must delete this files and path.
